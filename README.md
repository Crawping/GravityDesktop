# Gravity Desktop  

### 使用方法
GravityDesktop.slnを開く  
Box2D->DirectXTK_Desktop_2017->LocalHook->mainの順にコンパイル  
Escで終了  

### 使用ライブラリ 
Box2D:[https://github.com/erincatto/Box2D](https://github.com/erincatto/Box2D)
